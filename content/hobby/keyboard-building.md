## きっかけ

エンジニアとしてコードを書く時間が長く、キーボードにこだわりたいと思ったのがきっかけです。

## 用意したもの

- キットボード（Corne Cherry v3）
- Cherry MX 赤軸スイッチ × 42個
- キーキャップ
- ハンダゴテとハンダ

## 組み立て手順

### 1. ダイオードの取り付け

まず基板にダイオードをハンダ付けします。方向に注意が必要です。

```
[基板]
  |
  ダイオード（方向: カソード → マーク側）
  |
[スイッチ]
```

### 2. スイッチの取り付け

ダイオードが完了したら、スイッチをはめ込んでハンダ付けします。

### 3. ファームウェアの書き込み

QMK Firmwareを使ってカスタムキーマップを設定します。

```bash
# QMKのセットアップ
qmk setup

# ファームウェアのコンパイルと書き込み
qmk flash -kb crkbd -km default
```

## 完成！

初めてのハンダ付けでしたが、なんとか完成しました。打鍵感が最高で、毎日のコーディングが楽しくなりました。
